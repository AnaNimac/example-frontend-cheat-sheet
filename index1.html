Što je HTML?

  • HTML je kratica za HyperText Markup Language, što prevedeno znači prezentacijski jezik za kreiranje internet stranica.
  • Strogo gledano HTML nije programski jezik, nego jednostavan jezik za oblikovanje internet stranica.
  • Danas se HTML uglavnom koristi za kreiranje “kostura” web stranice, dok se oblikovanje web stranice radi preko CSS-a.
